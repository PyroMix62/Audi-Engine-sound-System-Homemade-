# Audi Engine Sound System Homemade 


L’ESS (Engine Sound System) est un système qui permet de simuler / amplifier les sons de l’échappement sur les modèles S de chez Audi.
Les V6 tdi peuvent ainsi émettre via des woofers placés dans la ligne d’échappement des son proches de celui d’un V8 essence.

Le système relativement simple coûte relativement chers chez audi ~2500€.
Il consiste en un calculateur / Ampli qui en fonction du régime moteur, charge, … module le signale envoyé sur un ou deux woofers

Le projet consiste à reproduire ce system à l’aide d’arduino / ESP32 d’un ampli et de deux Woofer (voir 4) de 100W

Il se décompose en deux parties
- La première l’acquisition du signal sur un system ESS original Audi et de déterminer quels paramètres modulent le signal via le canbus
- La seconde en générant le signal à l’aide d’un arduino et d’un ampli de basse en fonction du canbus ou du Kline sur des modèles plus anciens
